## What is DevOps?

DevOps is a culture and methodology that helps deliver software faster and more efficiently using:
- automation
- code quality maintenance
- continuous monitoring (observability)
- continuous testing

---

## Why DevOps?

Before DevOps, software deployment involved many manual steps and multiple teams, which often caused delays and errors.

DevOps helps automate processes, reduce manual effort, improve collaboration, and deliver applications faster and more reliably.
